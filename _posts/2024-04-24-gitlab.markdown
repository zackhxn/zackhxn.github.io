> “Yeah It's on. ”

# 克隆仓库
```
git clone git@github.com:zackhxn/zackhxn.github.io.git
```
# 更新本地仓库
```
git pull origin master
```
# 查看本地分支和远程分支
```
git branch
git branch -r
```
# 创建分支
```
git branch <分支名>
```
# 切换分支
```
git checkout <分支名>
```
# 查看修改了哪些
```
git status
```
```
git add .
git commit -m ""
```
# 推送新分支
```
git push origin <分支名>
```
